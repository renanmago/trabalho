O que é Responsividade ?


Definição

Responsividade é a capacidade de uma página se adaptar a diferentes tamanhos de tela, mantendo a usabilidade.

Design Adaptativo

É o processo de criar páginas responsivas com uso de tecnologias como media queries e class conteiner que e proveniente do bootstrap.

Mobile First

É a abordagem de criar primeiro uma versão para dispositivos móveis e depois aprimorá-la para desktop.



Qual a importancia?

  • Maior satisfação:usuarios tendem a abondonar sites não responsivos pela dificuldade de utilização
  • Aumento de Acesso no Mobile
  • Melhor classificação
  • Mais acessos
